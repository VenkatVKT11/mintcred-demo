## 📖 About The Project

MintCred is a premium, high-performance **"Coming Soon"** landing page built for a modern fintech loan startup. Built entirely in React, it is designed from the ground up to establish immediate trust with potential investors and business owners. 

The clean UI is inspired by industry-leading financial apps (like Stripe and Razorpay), utilizing a crisp white theme, soft glassmorphism card layouts, and subtle, highly-professional interactive animations.

### ✨ Key Features
- **Flawless Responsiveness:** Perfect formatting across desktop, tablet, and mobile displays.
- **Dynamic Previews:** Smooth entrance and hover-state animations powered by Framer Motion.
- **Interactive EMI Calculator:** An engaging drag-to-calculate loan visualizer that estimates monthly ROI payments.
- **Async Waitlist Capture:** Interactive email form snippet with loading & success states to capture user interest.
- **SEO Ready:** Highly optimized meta-tags and Open Graph tags for robust link sharing on platforms like LinkedIn, WhatsApp, and Twitter.

- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Deployment:** GitHub Pages

## 💻 Running it Locally

If you'd like to test the site on your local machine:

1. Clone this repository
   ```bash
   git clone https://github.com/VenkatVKT11/mintcred.git
   ```
2. Navigate into the folder
   ```bash
   cd mintcred
   ```
3. Install the dependencies
   ```bash
   npm install
   ```
4. Start the dev server
   ```bash
   npm run dev
   ```
5. Open `http://localhost:5173` in your browser.
